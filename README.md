# WalletAdmin

WalletAdmin is an ASP.NET Core MVC application for managing customers, wallets, and financial transactions.

## Features
- User authentication & authorization
- Customer management
- Wallet creation and balance tracking
- Credit & debit transactions
- Transaction history per wallet
- Dashboard overview

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- PostgresSQL
- Bootstrap

## Setup Instructions
1. Clone the repository
2. Update the connection string if needed
3. Run database migrations
4. Start the application
